
# 21世纪资本论

# 导言

财富分配到底会被少数人掌握，还是会随着竞争和技术进步逐渐降低社会各阶层之间的不平等程度？

每个人都会对所处时代的财富收入水平有自己的直观认识。这不只是经济学家的任务。

人口的持续增长、农村人口的外迁以及工业革命的来临等剧变是如何影响财富分配、社会结构以及欧洲社会政治稳定的呢？

人口增长带来的恐惧。

李嘉图提出稀缺性原则。

马克思提出无限积累原则。资本不可逆转地不断积累并最终掌握在一小部分人手中。

库兹涅茨曲线：在工业化和经济发展地进程中，收入不平等必然会出现先扩大后缩小地过程。

本书数据的两大来源：关于收入不平等和收入分配的数据；关于财富分配和财富-收入关系的数据

* 财富分配的历史总是深受政治影响，是无法通过纯经济运行机制解释的。
* 有一个强大的机制在交替性的推动收入与财富的趋同于分化。这个主要的力量是知识和技能的扩散。
* 当经济增长疲软而资本回报高的时候，在财富积累和集中的过程中会伴随着一系列分化力量，他们是导致长期财富分配不平等的主要因素。
* **分化的根本力量：资本收益率>经济增长率**

# 第一部分 收入和资本

## 收入和产出

国民收入：一年内一国居民可得的所有收入之和。

国内生产总值减去生产所用的资本折旧

国民收入 = 国内产值 + 国外净收入

全球收入 = 全球产出

国民收入 = 资本收入 + 劳动收入

本书提到的资本不包括人力资本。

国民财富 = 私人财富 + 公共财富 = 国民资本 = 国内资本 + 净国外资本

国民财富几乎由私人财富构成

资本/收入比：收入是流量（一般是一年）、资本是存量

资本主义第一基本定律：国民收入中资本收入的比重 = 资本收益率 * 资本/收入比
